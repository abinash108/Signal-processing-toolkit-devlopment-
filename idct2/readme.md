# idct2
## Description
Compute the inverse 2-D discrete cosine transform of matrix x.

## Calling Sequence

- `idct2(x)`

- `idct2(x,m,n)`
- `idct2(x,[m,n])`
## Parameters
- `x` : A Matrix
- `[m,n]` : zero-pads or crops the matrix A to size m-by-n before applying the inverse transformation.
